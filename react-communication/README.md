# React 组件通信

## 常见的组件通信方式

Props(p->c), 回调函数(c->p)，Context(multi)

父子传值
父组件通过props传递给子组件，子组件通过回调函数传递给父组件

兄弟组件传值
通过父组件作为中间层，将数据传递给兄弟组件

跨级组件传值
通过Context跨级传递数据

事件监听传值
